# Coze API 成本修正说明（海外版美元计费）

> **修正版本**: v1.1
> **修正时间**: 2025-01-27
> **修正原因**: Coze 使用海外版，Token 使用美元结算

---

## 🔴 重要修正

### 原计算错误（使用了国内版定价）

❌ **错误**: 使用了 Coze 国内版定价（人民币直接计费）
- 前 1M Tokens: 免费
- 1M - 10M: ¥0.03/千Token
- 10M+: ¥0.02/千Token
- **年度成本**: ¥1,078

---

### 正确计算（Coze 海外版定价）

✅ **正确**: 使用 Coze 海外版定价（美元计费 + 汇率换算）

#### Coze 海外版 API 定价

| 使用量 | 价格（美元） | 换算人民币（汇率 1:7.2） |
|-------|------------|----------------------|
| 前 1M Tokens | 免费 | 免费 |
| 1M - 10M Tokens | **$0.004/千Token** | ¥0.0288/千Token |
| 10M+ Tokens | **$0.003/千Token** | ¥0.0216/千Token |

**数据来源**: Coze 海外版官网（2025年1月）

---

## 📊 成本对比（年度，50.4M Tokens）

### 场景：Fiido 12,000 单/年

| 项目 | 国内版（错误） | 海外版（正确） | 差异 |
|-----|-------------|-------------|------|
| **Token 消耗** | 50.4M Tokens/年 | 50.4M Tokens/年 | 相同 |
| **美元成本** | N/A | **$157.2/年** | - |
| **人民币成本** | ¥1,078 | **¥1,132**（$157.2 × 7.2） | +¥54 |
| **差异比例** | - | - | **+5%** |

**结论**: 海外版比国内版贵 **5%**（¥54/年），差异不大。

## 💰 修正后的成本结构

### 模式 1: SaaS 托管（你承担所有成本）

| 成本项 | 修正前 | 修正后 | 说明 |
|-------|-------|-------|------|
| Coze API | ¥1,078 | **¥1,132** | +¥54/客户/年 |
| 云服务器 | ¥1,260 | ¥1,260 | 不变 |
| 翻译/视频 | ¥3,600 | ¥3,600 | 不变 |
| 运维人力 | ¥1,500 | ¥1,500 | 不变 |
| **总成本** | ¥7,438 | **¥7,492** | +¥54 |

**毛利润变化**（企业版 ¥85,200/年）:
- 修正前毛利润: ¥77,762（91.3%）
- 修正后毛利润: **¥77,708**（**91.2%**）
- 差异: -¥54（-0.1%）⬅️ **影响很小**

---

### 模式 2: 定制部署（客户承担成本）

| 成本项 | 修正前 | 修正后 | 承担方 |
|-------|-------|-------|--------|
| 定制费（首年） | ¥108万 | ¥108万 | 客户付给你 |
| 年服务费 | ¥20万 | ¥20万 | 客户付给你 |
| 服务器 | ¥7,700 | ¥7,700 | **客户自己** |
| Coze API | ¥1,078 | **¥1,132** | **客户自己**（美元结算） |
| 翻译/视频 | ¥3,600 | ¥3,600 | **客户自己** |
| **客户运营成本** | ¥12,378 | **¥12,432** | +¥54 |

**对你的影响**: **无影响**（成本由客户承担）

**对客户的影响**:
- 第1年: ¥128万（付给你）+ ¥1.24万（运营成本）= ¥129.24万
- 修正后: ¥128万 + **¥1.2432万** = **¥129.2432万**
- 差异: **+¥54/年**（0.04%）⬅️ **可忽略不计**

---

## ✅ 关键结论

### 1. 成本差异很小

| 对比维度 | 差异 | 影响 |
|---------|------|------|
| **单客户年度成本** | +¥54 | 极小（5%） |
| **SaaS 模式毛利率** | -0.1% | 可忽略 |
| **定制模式你的收入** | ¥0 | 无影响 |
| **定制模式客户成本** | +¥54/年 | 可忽略（0.04%） |

---

### 2. 推荐策略不变

**仍然推荐**: 定制部署 + 年订阅混合模式

**理由**:
- ✅ Coze API 成本由客户承担（无论海外版还是国内版）
- ✅ 你的收入不受影响（¥108万 + ¥20万/年）
- ✅ 客户增加的成本可忽略（¥54/年 vs ¥128万总投资）

---

### 3. 对客户的影响

**告知客户**:
- 使用 Coze 海外版 API（美元结算）
- 年度 Token 成本: **$157**（约 **¥1,132**）
- 需要支持美元支付（信用卡/PayPal）

**可选方案**（如果客户介意美元支付）:
- 改用 Coze 国内版（如果有）
- 年度成本降低至 ¥1,078（节省 ¥54）
- 但需要评估功能差异

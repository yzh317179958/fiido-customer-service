# Coze知识库自动填充工具 - 使用说明

## 📋 功能说明
自动将Excel中的问答数据（141行）填充到Coze知识库网页表单中

## 🚀 使用步骤

### 方法1: 浏览器控制台（推荐）✨

1. **打开目标网页**
   - 在Firefox浏览器打开Coze知识库页面
   - URL: https://www.coze.com/space/7568762113083064337/knowledge/7576922848696532997/upload

2. **打开浏览器控制台**
   - 按 `F12` 键打开开发者工具
   - 或者右键页面 → 选择"检查元素" → 切换到"控制台(Console)"标签

3. **复制并运行脚本**
   - 打开文件: `/home/yzh/AI客服/鉴权/fill_coze_auto.js`
   - 全选复制整个文件内容 (Ctrl+A, Ctrl+C)
   - 粘贴到浏览器控制台 (Ctrl+V)
   - 按 `Enter` 键运行

4. **等待自动填充**
   - 脚本会自动逐行填充数据
   - 控制台会显示进度: `[1/141] 填写: C11pro车架是铝合金吗...`
   - 预计耗时: 约1-2分钟（141行 × 每行0.8秒）

5. **手动提交**
   - 填充完成后，控制台显示: `🎉 填充完成！`
   - **手动检查**最后几行数据是否正确
   - 点击页面的 **"Confirm"** 按钮提交

## ⚠️ 注意事项

1. **不要关闭页面**: 脚本运行期间请勿刷新或关闭页面
2. **不要切换标签**: 保持在当前标签页，否则可能中断
3. **等待完成**: 看到"🎉 填充完成！"后再进行其他操作
4. **手动检查**: 建议滚动检查前几行和最后几行数据是否正确
5. **备份数据**: 提交前建议截图保存

## 📊 数据说明

- **数据行数**: 141行（不包括标题行）
- **列1**: 问题 (填入"Index string"字段)
- **列2**: 问题回复 (填入"2 String"字段)

## 🛠️ 故障排查

### 如果填充失败：

1. **检查网页是否加载完成**
   - 确保"Add value"按钮可见

2. **检查是否需要重新登录**
   - 登录可能过期，请刷新页面重新登录

3. **查看控制台错误信息**
   - 红色错误提示会显示具体问题

4. **重新运行脚本**
   - 按 `Ctrl+L` 清空控制台
   - 重新粘贴脚本运行

### 如果部分数据填充失败：

- 控制台会显示: `❌ 失败: X 行`
- 记录失败的行号
- 手动补充这些行的数据
